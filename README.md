# 🗑️ IoT-Based Smart Waste Management System

## 📌 Overview
An IoT-based Smart Waste Management System that monitors garbage bin fill levels in real time using sensors. The system sends alerts when the bin reaches a threshold level, helping optimize waste collection and prevent overflow.

## ✨ Features
- Real-time garbage level monitoring
- Ultrasonic sensor for fill-level detection
- GPS location tracking
- GSM-based SMS alerts
- LCD display for status updates
- ESP32-based implementation

## 🛠️ Hardware Used
- ESP32
- HC-SR04 Ultrasonic Sensor
- NEO-6M GPS Module
- SIM900 GSM Module
- 16x2 I2C LCD
- 5V Power Supply

## 💻 Software Used
- Arduino IDE
- Wokwi Simulator
- Embedded C/C++
